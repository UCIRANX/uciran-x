## 🆕 تغییرات نسخه‌ی جدید (worker-1.js)

نسخه‌ی `worker-1.js` نسبت به `worker.js` دو تغییر اصلی دارد:

1. **حذف مدل DeepSeek V4 Flash (OpenModel)** — این مدل و تمام کدهای مربوط به سرویس OpenModel (متغیر `OPENMODEL_API_KEY`، تابع `callOpenModel`، و مرحله‌ی مربوطه در `FALLBACK_ORDER`) به‌طور کامل حذف شده‌اند. در نتیجه ترتیب fallback از ۷ مرحله به ۶ مرحله کاهش پیدا کرده و دیگر نیازی به تنظیم `OPENMODEL_API_KEY` نیست.

2. **تشخیص خودکار سایز عکس + کوچک‌سازی هوشمند برای صرفه‌جویی در Neuron** — قابلیت جدیدی به دستورهای `/draw` و `/edit` اضافه شده:
   - اگر کاربر داخل متن پرامپت یک الگوی سایز مثل `1024x720` یا `1024*720` بنویسد، ربات با تابع جدید `extractSizeFromPrompt()` آن را از متن پیدا و حذف می‌کند (تا وارد پرامپت مدل تصویرساز نشود).
   - سپس تابع جدید `scaleDimensions()` همان نسبت تصویر (aspect ratio) را حفظ می‌کند، اما بزرگ‌ترین ضلع را همیشه به حداکثر ۵۰۴ پیکسل (مضرب ۸، هدفش پایین نگه‌داشتن مصرف Neuron مدل FLUX روی پلن رایگان Cloudflare Workers AI است) محدود می‌کند.
   - در دستور `/edit`، این نسبت به‌صورت خودکار از ابعاد واقعی عکسی که تلگرام برمی‌گرداند گرفته می‌شود، بدون نیاز به وارد کردن سایز توسط کاربر.
   - اگر هیچ سایزی مشخص نشود، پیش‌فرض روی ۵۰۴×۵۰۴ تنظیم می‌شود (به‌جای ۱۰۲۴×۱۰۲۴ در نسخه‌ی قبلی `worker.js`).

> 💡 هدف این تغییر، کاهش مصرف سهمیه‌ی Neuron مدل FLUX روی پلن رایگان Cloudflare است تا محدودیت روزانه دیرتر پر شود؛ کیفیت تصویر نهایی همچنان با حفظ نسبت درخواستی کاربر تولید می‌شود.

## 👇 آموزش ویدیویی راه‌اندازی ربات هوش مصنوعی تلگرام 👇
(https://t.me/UCIRANIR/33001)

### پیش‌نیازها

- یک حساب [Cloudflare](https://dash.cloudflare.com/) (پلن رایگان کافی است)
- یک بات تلگرام ساخته‌شده با [@BotFather](https://t.me/BotFather) و توکن آن
- (اختیاری) کلید API از [Google AI Studio](https://aistudio.google.com/) برای Gemini
- (اختیاری) کلید API از [Groq Console](https://console.groq.com/) برای Groq Llama 4 Scout

### مراحل نصب

**۱. ساخت Worker**
کد `worker-1.js` (آخرین نسخه) را در یک Worker جدید در داشبورد Cloudflare قرار دهید، یا با Wrangler دیپلوی کنید.

**۲. اتصال KV Namespace**
یک KV Namespace بسازید و آن را با نام متغیر `KV` به Worker بایند کنید. این برای ذخیره‌ی حافظه‌ی مکالمه، انتخاب مدل کاربر و شمارنده‌ی پیام‌ها استفاده می‌شود.

**۳. فعال‌سازی Workers AI**
باینداینگ Workers AI را با نام `AI` به Worker اضافه کنید (از بخش Settings → Bindings).

**۴. تنظیم متغیرهای محیطی**

| متغیر | اجباری | توضیح |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | ✅ | توکن ربات از BotFather |
| `WORKER_DOMAIN` | ✅ | آدرس Worker، مثلاً `my-bot.username.workers.dev` |
| `GEMINI_API_KEYS` | ❌ | یک یا چند کلید Gemini با کاما جدا شده (برای جستجوی آنلاین) |
| `GROQ_API_KEYS` | ❌ | یک یا چند کلید Groq با کاما جدا شده (از [console.groq.com](https://console.groq.com/) — رایگان) |

**۵. وصل کردن وب‌هوک**
بعد از دیپلوی، آدرس زیر را یک‌بار در مرورگر باز کنید:

```
https://<WORKER_DOMAIN>/setWebhook
```

اگر پاسخ `"ok": true` دریافت کردید، ربات آماده‌ی استفاده است. 🎉

## 💬 دستورات ربات

| دستور | عملکرد |
|---|---|
| `/start` | شروع و نمایش راهنمای کلی |
| `/help` | راهنمای کامل دستورات |
| `/clear` | پاک کردن حافظه‌ی مکالمه |
| `/models` | نمایش لیست مدل‌های فعال |
| `/engine` | انتخاب مدل هوش مصنوعی از منو |
| `/online [سوال]` | پاسخ با جستجوی آنلاین (Gemini) |
| `/draw [توضیح]` | ساخت عکس از روی متن (مدل FLUX.2 Klein) |
| `/edit [توضیح]` | ویرایش عکس — روی یک عکس ریپلای کن و دستور را بنویس |

علاوه بر این، ارسال مستقیم عکس، ویدیو (تا حداکثر حجم تعیین‌شده) یا فایل متنی/کد نیز به‌صورت خودکار پردازش می‌شود.

**🆕 تعیین سایز تصویر در `/draw`:** می‌توانید داخل توضیح، سایز دلخواه را هم بنویسید، مثلاً:
```
/draw یک منظره‌ی کوهستانی 1024x720
```
ربات نسبت تصویر (۱۰۲۴ به ۷۲۰) را حفظ می‌کند اما برای صرفه‌جویی در مصرف Neuron روی پلن رایگان، بزرگ‌ترین ضلع را حداکثر ۵۰۴ پیکسل می‌سازد. در `/edit` هم این نسبت به‌طور خودکار از روی عکس اصلی گرفته می‌شود و نیازی به وارد کردن سایز نیست.

## ⚙️ تنظیمات قابل شخصی‌سازی

در ابتدای فایل `worker.js`:

```js
const CHANNEL      = '@uciranir';      // یوزرنیم کانال
const CHANNEL_LINK = 'https://t.me/uciranir';
const SHOW_CH_EVERY = 15;              // نمایش تبلیغ کانال هر چند پیام
const MAX_HISTORY   = 6;               // تعداد رفت‌وبرگشت‌های ذخیره‌شده در حافظه
const MSG_LIMIT     = 4000;            // حداکثر طول پیام ارسالی
const MAX_VIDEO_MB  = 19;              // حداکثر حجم ویدیوی قابل پردازش
```

> ⚠️ **نکته‌ی مهم درباره‌ی Gemini:** کلیدهای متعلق به یک حساب/پروژه‌ی گوگل، سهمیه‌ی (Quota) مشترک دارند. برای افزایش واقعی سهمیه، هر کلید باید از یک حساب گوگل جدا گرفته شود.
